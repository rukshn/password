Password best practises

  • Do not use the same password on many websites
  • If you can't remember passwords use a password manager
  • Make sure your password is long and has symbols, numbers and both capital and simple letters
  • No password is safe, use two factor authentication when available
  • Do not enter your passwords on any unknown websites
  • Do not use any words, birthdays, mobile numbers, names as your password
